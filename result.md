## 扇形

`θ`: 扇形角度

`dθ`: 顺时针角度偏移

### 单向

单表达式（θ <= pi）：

```
((-sin(-θ/2) * (x(C) - x(B)) + cos(-θ/2) * (y(C) - y(B))) * (x - x(B)) - (cos(-θ/2) * (x(C) - x(B)) + sin(-θ/2) * (y(C) - y(B))) * (y - y(B)))((-sin(θ/2) * (x(C) - x(B)) + cos(θ/2) * (y(C) - y(B))) * (x - x(B)) - (cos(θ/2) * (x(C) - x(B)) + sin(θ/2) * (y(C) - y(B))) * (y - y(B))) <= 0
```

使用中间参数：

```
u1 = 旋转(向量(B,C),((θ)/(2))-dθ)
u2 = 旋转(向量(B,C),-((θ)/(2))-dθ)
a: y(u1) (x-x(B))-x(u1) (y-y(B)) ≥ 0
b: y(u2) (x-x(B))-x(u2) (y-y(B)) ≤ 0
if(θ ≤ 180°, a ∧ b, a ∨ b)
```

使用中间参数：

## 地火

`dθ`: 地火方向

### 单向地火

```
u = {{sin(dθ)}, {-cos(dθ)}}
center = B + t d u
```

圆:
```
(x - x(center)) ^ 2 + (y - y(center)) ^ 2 ≤ r^2
```

箭头：
```
Fan((x(center) + 0.8 r x(u), y(center) + 0.8 r y(u)), B, 90°, 0°) ∧ Fan((x(center) + 0.6 r x(u), y(center) + 0.6 r y(u)), (x(center) + r x(u), y(center) + r y(u)), 270°, 0°) ∧ (x - x(center) - 0.45 r x(u)) x(u) + (y - y(center) - 0.45 r y(u)) y(u) ≥ 0 ∨ (x - x(center)) ^ 2 + (y - y(center)) ^ 2 ≤ 0.005 r^2
```
